Trail Making Test Module Documentation
=====================================

The Trail Making Test (TMT) module implements a digital version of the standardized neuropsychological assessment tool. It follows a clean architecture approach with three main layers: Presentation, Domain, and Data. Each layer has specific responsibilities and components dedicated to implementing a valid and reliable cognitive assessment.
